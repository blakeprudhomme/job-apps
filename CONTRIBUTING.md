# Contributing to job-apps

Thanks for your interest in contributing 🙌

This project aims to simplify and automate the job application process — from tracking roles to eventually assisting with applications and workflows.

---

## 🧭 Project Direction

This is an early-stage project. Contributions are especially valuable in shaping:

- Job data ingestion (APIs, scraping, imports)
- Application tracking workflows
- UI/UX for managing job pipelines
- Automation features (resume tailoring, scoring, etc.)

If you're unsure where to start, open an issue — happy to collaborate.

---

## 🚀 Getting Started

1. Fork the repo
2. Clone your fork

   git clone https://github.com/<your-username>/job-apps.git

3. Create a branch

   git checkout -b feature/your-feature-name

4. Run the project (see README for setup)

---

## 🧱 Project Philosophy

- **Practical > Perfect**
- **Readable > Clever**
- **Iterate fast**

Avoid over-engineering — build things that are easy to extend.

---

## 🧩 Contribution Areas

We especially welcome:

### 1. Data Sources
- Job board integrations
- Importers (CSV, LinkedIn exports, etc.)

### 2. Core Features
- Job tracking models
- Status workflows (Applied → Interview → Offer)
- Notes / metadata

### 3. Automation
- Job parsing
- Resume matching/scoring
- AI-assisted workflows

### 4. UI / UX
- Dashboards
- Filtering / sorting
- Pipeline views (kanban-style)

---

## 🧪 Testing

- Add tests for new features when possible
- Don’t break existing functionality

---

## 📝 Commit Style

Examples:

- feat: add job import from CSV
- fix: handle missing company field
- docs: update contributing guide

---

## 🔁 Pull Requests

Please:

- Keep PRs small and focused
- Explain **why** the change exists
- Link to issues if applicable

---

## 💬 Before You Start

If you're planning something non-trivial:

👉 Open an issue first to discuss direction

This helps avoid duplicated work and keeps the project cohesive.

---

## 🤝 Code of Conduct

Be respectful, constructive, and helpful.

---

Thanks again for contributing ❤️
